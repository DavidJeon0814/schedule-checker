# schedule-checker backend

## installation

### [Visual Studio Code](https://code.visualstudio.com)

#### Visual Studio Code EXTENSIONS

- indent-rainbow

### [Node.js](https://nodejs.org/ko)

#### check version

```console
$ node -v
v16.13.2
```

### nodejs

```console
$ npm init -y
$ npm i
$ node server.js
```

### if local is changed or after git clone

```
$ npm i
```

### run vue (alternative)

```
$ 1startup-be.bat
```

### build vue (alternative)

```
$ 2build-be.bat
```

### add modules

#### express

```console
$ npm i express
```

#### nodemon

```console
$ npm i nodemon -D
```
