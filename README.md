# vue

## installation

### [Visual Studio Code](https://code.visualstudio.com)

#### Visual Studio Code EXTENSIONS

- Vue Language Features (Volar)

- HTML CSS Support

- Auto Rename Tag

- indent-rainbow

### [Node.js](https://nodejs.org/ko)

#### check version

```console
$ node -v
v16.13.2
```

### vue with vite

```console
$ npm init vite@latest -y
√ Project name: »client
√ Select a framework: »vue
√ Select a variant: »vue
$ cd client
$ npm i
$ npm run dev
```

### run vue (alternative)

```
$ 1startup-client.bat
```

### add modules

#### vue-router

```console
$ npm i vue-router@next
```

#### vue-sweetalert2

```console
$ npm i vue-sweetalert2
```
